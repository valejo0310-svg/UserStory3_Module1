# User Story Week 3

A python program made to register the sales, and modify them while also storing them on a CSV, so all the data can be used later after on.

## ► What does it do?
- It allows the client to add products while the program is running.
- The program shows the purchases and allows the user to keep track .
- Let´s the client search for the products previously
- Allows the client to update the products added (new prices and new .
- Eliminates the products the client wishes.
- Shows the statistics of the purchase (Total cost, highest on stock and the highest price).
- Allows the client to store all the information on CSV to access the purchase later.
- Being able to load all the CSV of the progress.
- Allows the client to exit whenever they want.
